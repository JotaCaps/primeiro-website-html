Textos

São basicamente as tags relacionadas a formatação de textos, como exemplos temos de "H1" a "H6" pra titulos e suas hierarquias, "p" para parágrafos, e diversas outras para formataçaõ, como "abbr" pra abreviacões, "strong" para destacar os textos, "i" para Ítalico e por aí vai

Listas ordenadas e não ordenadas

São as tags "ol" para listas ordenadas e "ul" para listas não ordenadas, lembrando que para colocar os itens nas listas é usada a tag filho "li" para cada item

Links

Com a tag "a" você pode ancorar links da web ou links externos a um texto selecionado
